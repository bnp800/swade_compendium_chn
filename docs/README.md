## 快速开始

### 我想参与翻译

1. 访问 [Weblate 翻译平台](http://150.109.5.239/engage/swade/)
2. 注册账号并开始翻译
3. 详细流程请参考 [翻译工作流文档](./translation-workflow.md)

### 我想导出新的源文件

1. 安装 Translation Generator 模块
2. 在 FoundryVTT 中导出 Compendium
3. 详细步骤请参考 [Translation Generator 使用指南](./translation-generator-guide.md)

### 我想在本地测试翻译

1. 配置本地 FoundryVTT 环境
2. 链接翻译模块目录
3. 详细配置请参考 [本地开发环境配置指南](./local-dev-guide.md)
