# Product Overview

This is a Chinese (Simplified) translation module for the SWADE (Savage Worlds Adventure Edition) tabletop RPG system in Foundry VTT.

## Purpose
- Provides Chinese translations for SWADE compendium packs (edges, hindrances, powers, equipment, bestiary, etc.)
- Requires the Babele module for translation injection
- Supports both SWADE core rules and SWPF (Savage Worlds Pathfinder) content

## Target Users
- Chinese-speaking Foundry VTT users playing SWADE
- Translators contributing to the localization effort via Weblate

## Module ID
`swade_compendium_chn`

## Compatibility
- Foundry VTT: v11-v13
- SWADE System: v5.0.0+
- Requires: Babele module
